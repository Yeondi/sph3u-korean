종합 회로 조사 접근 가능 버전

이 문서는 회로 시뮬레이션을 완료하는 데 대한 접근 가능한 대안으로 사용할 수 있습니다.

기구

회로 구성 키트: DC - 가상 실험실을 사용하여 회로도와 일치하는 회로를 구성했습니다. 9.0볼트 전지(battery)에서 시작하여, 전선이 전지의 양극 단자에서 열린 스위치(switch)로 연결됩니다.

스위치 이후 두 개의 병렬(parallel) 경로가 분기됩니다. 각 경로에는 전류계(ammeter) 하나가 저항기(resistor) 하나와 직렬(series)로 연결되어 있습니다. 두 경로는 저항기 이후 합류하여 전지의 음극 단자로 돌아갑니다.

처음에 두 저항기의 저항(resistance)은 10.0옴입니다.

결과

스위치를 닫은 후, 두 저항기가 모두 10.0옴으로 설정된 상태에서 두 전류계 모두 0.90암페어의 전류(current)를 보여줍니다.

저항기 중 하나를 5.0옴으로 변경하면, 해당 전류계가 1.80암페어로 변합니다.

같은 저항기를 20.0옴으로 변경하면, 해당 전류계가 0.45암페어로 변합니다.

같은 저항기를 25.0옴으로 변경하면, 해당 전류계가 0.36암페어로 변합니다.

같은 저항기를 30.0옴으로 변경하면, 해당 전류계가 0.30암페어로 변합니다.

같은 저항기를 40.0옴으로 변경하면, 해당 전류계가 0.22암페어로 변합니다.
