과제 3: 무엇을 알고 있나요?

다음 문제를 풀어 보세요. 막히면 힌트를 확인하세요. 풀이 예시와 비교하여, 자신의 풀이가 정확하고 명확하며 완전한지 되돌아보세요.

  1. 280.0 Hz 부저가 파장이 1.21 m인 음파를 만들어 냅니다. 소리가 전달되는 공기의 온도는 얼마입니까?

힌트

  1. 파동의 기본 방정식 v   = f λ 을 먼저 사용하여 속력을 구하세요.

  2. 공기 중 음속 공식을 사용하세요
    ( v = 332   m / s + ( 0.59   m / s   ° C ) T )
    여기서 T는 온도이고 v는 음파의 속력입니다.

풀이 보기

주어진 값:

f = 280.0 Hz

λ = 1.21 m

구하는 값:

T =  ?

공식:

v = f λ 은 파동의 기본 방정식입니다.

음속 공식도 필요합니다

v s o u n d = 332  m / s + ( 0.59  m / s ° C ) T

풀이:

먼저 음속을 구합니다.

v = ( 280.0 Hz ) ( 1.21 m )

v = 338.8  m / s

그런 다음 음속과 온도의 관계식을 사용하여 온도를 구합니다.

v s o u r c e = 332  m / s + ( 0.59  m / s ° C ) T

338.8  m / s = 332  m / s + ( 0.59  m / s ° C ) T

6.8  m / s = ( 0.59  m / s ° C )

6.8  m / s 0.59  m / s ° C = T

11.53 ° C = T

결론:

공기의 온도는 11.5 °C입니다. (주어진 값이 유효 숫자 4자리와 3자리이므로, 유효 숫자 3자리로 나타냅니다.)

  1. 진동수가 420 Hz인 소리굽쇠가 공기 중에서 파장 0.82 m의 소리를 냅니다. 공기의 온도가 올라가면 파장은 어떻게 되나요: 증가, 감소, 또는 변화 없음?

힌트

  1. 온도가 속력에 어떤 영향을 미칠지 생각해 보세요.
  2. 파동의 기본 방정식을 생각해 보세요
    ( v = f λ )
    v가 증가하지만 f가 일정하다면, 파장은 어떻게 되어야 할까요?

풀이 보기

공기 중 음속 공식을 사용하면, ( v = 332  m / s + ( 0.59  m / s ° C ) T )

온도가 올라가면 속력이 증가한다는 것을 알 수 있습니다.

따라서 파동의 속력이 증가하고 있다는 것을 알 수 있습니다.

파동의 기본 방정식을 사용하면, ( v = f λ )

v가 증가하지만 f가 일정하다면, 파장(λ)은 증가해야 한다고 결론 내릴 수 있습니다.

  1. 다음 각각의 진동수와 주기를 구하세요:
    1. 전구가 1초 동안 60번 켜졌다 꺼집니다.
    2. 벌새가 6초 동안 날개를 120번 칩니다.

힌트

  1. 주기 공식은:

    T =  total time   number of cycles 

    또는

    T = t N

    여기서 N은 진동 횟수이고 t는 시간입니다. a)에서는 60번, b)에서는 120번 진동합니다.

  2. 주기를 알면 다음 공식을 사용할 수 있습니다

    f = 1 T

    이를 사용하여 진동수를 구합니다.

풀이 보기 1

주어진 값:

N = 60   t = 1.0  s

구하는 값:

f =  ?

공식:

T = t N

풀이:

T = 1 s 60 = 0.0167  s

f = 1 T = 1 0.0167  s = 60  Hz

결론:

주기는 0.02 s이고 진동수는 60 Hz입니다.

시간이 1 s이므로, 계산 없이도 진동수를 바로 말할 수 있습니다. 결론은 유효 숫자 1자리로 나타냅니다. "1 s"가 유효 숫자 1자리이기 때문입니다.

풀이 보기 2

주어진 값:

N = 120 , t = 6.0  s

구하는 값:

T =  ?

f =  ?

공식:

T = t N

풀이:

T = 6  s 120 = 0.050  s

f = 1 T = 1 0.050  s = 20  H z

결론:

주기는 0.05 s이고 진동수는 20 Hz입니다.

  1. 직선 선로 위의 기차가 도로에 주차된 자동차 옆을 지나며 400.0 Hz의 경적을 울립니다. 기차는 20.0 m/s로 이동하고 있으며 온도는 10.0°C입니다. 기차가 다가올 때 자동차 안에 앉아 있는 사람이 듣는 겉보기 진동수를 구하세요.

힌트

  1. 먼저 다음 공식을 사용하여 온도로부터 음속을 구하세요.

    v = 332 m / s + ( 0.59 m / s ° C ) T

  2. 도플러 효과 공식에서 + 또는 – 부호 중 어느 것을 사용해야 할지 결정하세요.

    f 2 = f 1 ( v s o u n d v s o u n d ± v s o u r c e )

풀이 보기

주어진 값:

f 1 = 400.0 Hz

v s o u r c e = 20.0  m / s

T = 10.0 ° C

구하는 값:

f 2 =  ?

공식:

v s o u r c e = 332  m / s + ( 0.59  m / s ° C ) T

f 2 = f 1 ( v s o u n d v s o u n d − v s o u r c e )

풀이:

먼저 온도에 따른 음속을 구합니다.

v s o u r c e = 332  m / s + ( 0.59  m / s ° C ) T

v s o u r c e = 332  m / s + ( 0.59  m / s ° C ) ( 10.0 ° C )

v s o u r c e = 337.9  m / s

기차가 다가오고 있으므로 도플러 효과 공식에서 빼기를 사용합니다.

f 2 = ( 400  H z ) ( 337.9  m / s 337.9  m / s − 20  m / s )

f 2 = ( 400  H z ) (1.063)

f 2 = 425.2  H z

결론:

그 사람은 425 Hz의 진동수를 관측하게 됩니다.
