방사성 붕괴(Radioactive Decay)

이 시뮬레이션에서는 방사성 붕괴(radioactive decay)가 두 가지 변수 — 방사성 물질의 반감기(half-life)와 시료 내 방사성 원자의 수 — 에 의해 어떻게 영향을 받는지 탐구합니다.

활동 방법

슬라이더 막대를 사용하여 각 변수를 조정하세요. 하나 또는 두 변수를 조정한 후
[ Start ]를 클릭하면 붕괴 과정이 그래프로 표시됩니다. [ Reset ]을 클릭하면 화면이 초기화됩니다.

  1. 시료 내 방사성 입자의 수를 변경하면 반감기의 길이가 변하나요?
  2. 시료 내 방사성 입자의 수를 변경하면 모든 방사성 원자가 붕괴하는 데 걸리는 총 시간이 변하나요?
  3. 반감기를 변경하면 시료 내 모든 방사성 원자가 붕괴하는 데 걸리는 시간이 어떻게 변하나요?
